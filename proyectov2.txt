R.G.01 Administración y control de usuarios del sistema
Como empresa,
quiero contar con un sistema que permita la administración y control de los usuarios y sus niveles de acceso,
para garantizar una gestión segura, ordenada y diferenciada de clientes, técnicos y administradores.

R.F.01. Gestión de usuarios
Como administrador,
quiero registrar, actualizar y eliminar usuarios,
para mantener un control organizado de clientes y empleados.

R.F.02. Registro de usuario
Como cliente,
quiero registrarme en el sistema de manera sencilla y segura,
para poder acceder al servicio de alquiler.

R.F.03. Gestión de roles y permisos
Como administrador,
quiero asignar roles a los usuarios (cliente, técnico, administrador),
para definir los niveles de acceso y responsabilidad.

R.G.02. Gestión integral de la flota de vehículos
Como administrador del sistema,
quiero gestionar de forma integral los vehículos de movilidad personal registrados en la plataforma,
para mantener actualizada y controlada la flota disponible para el servicio de alquiler.

R.F.04. Registro de vehículos
Como administrador,
quiero registrar vehículos de movilidad personal con sus características,
para mantener actualizada la flota disponible para alquiler.

R.F.05. Actualización y baja de vehículos
Como administrador,
quiero actualizar la información o dar de baja vehículos,
para reflejar su disponibilidad y estado real.

R.G.03 Consulta de estaciones y disponibilidad de vehículos
Como usuario del sistema,
quiero consultar las estaciones de servicio junto con la disponibilidad de vehículos en cada una,
para poder elegir fácilmente el punto más conveniente para recoger o devolver un vehículo.

R.F.06. Consulta de estaciones
Como usuario,
quiero visualizar las estaciones disponibles con su ubicación y capacidad,
para elegir dónde recoger o dejar un vehículo.

R.F.07. Consulta de disponibilidad de vehículos
Como usuario,
quiero conocer el número de vehículos disponibles por estación,
para seleccionar la más conveniente para mí.

R.G.04 Gestión del proceso de alquiler y devolución de vehículos
Como usuario del sistema,
quiero realizar el proceso completo de alquiler y devolución de vehículos de forma rápida y sencilla,
para poder usar el servicio de movilidad cuando lo necesite y finalizarlo correctamente en una estación autorizada.

R.F.08. Alquiler de vehículos
Como usuario,
quiero alquilar un vehículo de manera rápida y sencilla,
para poder desplazarme cuando lo necesite.

R.F.09. Devolución de vehículos
Como usuario,
quiero devolver el vehículo en una estación autorizada,
para finalizar correctamente el alquiler y evitar cargos adicionales.

R.G.05.Gestión de pagos digitales y emisión de comprobantes
Como usuario del sistema,
quiero realizar los pagos de mis alquileres de forma digital y segura, con la emisión automática de comprobantes,
para completar las transacciones de manera confiable y disponer de un registro transparente de mis operaciones.

R.F.10. Pagos digitales
Como usuario,
quiero realizar el pago de mi alquiler de forma digital y segura,
para completar la transacción sin complicaciones.

R.F.11. Generación de comprobantes de pago
Como sistema,
quiero generar un comprobante digital del pago,
para garantizar transparencia y trazabilidad.

R.G.06 Supervisión y mantenimiento de la flota de vehículos

Como administrador del sistema,
quiero supervisar el estado, ubicación y mantenimiento de los vehículos,
para garantizar su correcto funcionamiento, planificar revisiones preventivas y mantener un historial técnico actualizado.

R.F.12. Control del estado de los vehículos
Como administrador,
quiero consultar el estado y ubicación de cada vehículo,
para planificar mantenimientos preventivos y resolver incidencias.

R.F.13. Registro de mantenimiento
Como administrador,
quiero registrar mantenimientos e incidencias de los vehículos,
para llevar un historial técnico actualizado.

R.F.14. Asignación de tareas de mantenimiento
Como administrador,
quiero asignar tareas de mantenimiento a los técnicos,
para organizar y distribuir eficientemente las reparaciones de los vehículos.

R.F.15. Ejecución y reporte de mantenimiento
Como técnico de mantenimiento,
quiero registrar la ejecución de las reparaciones e indicar el resultado,
para mantener actualizado el historial técnico y el estado real de los vehículos.


📊 Requisitos de Información (R.I.)

R.I.01. Información de usuarios
Como sistema,
quiero almacenar datos personales, credenciales y roles de los usuarios,
para permitir autenticación y gestión de perfiles.

R.I.02. Información de vehículos
Como sistema,
quiero registrar el tipo, modelo, número de serie, estado y ubicación de cada vehículo,
para mantener actualizada la flota de alquiler.

R.I.03. Información de estaciones
Como sistema,
quiero almacenar la ubicación, capacidad y disponibilidad de las estaciones,
para mostrar información precisa al usuario.

R.I.04. Información de alquileres
Como sistema,
quiero registrar el inicio, fin, usuario, vehículo y costo de cada alquiler,
para generar reportes y controlar el servicio.

R.I.05. Información de pagos
Como sistema,
quiero guardar los datos de cada transacción, método de pago y monto,
para garantizar la trazabilidad de los cobros.

R.I.06. Información de mantenimiento
Como sistema,
quiero registrar incidencias y acciones de mantenimiento,
para controlar el estado técnico de los vehículos.


⚙️ Requisitos No Funcionales (R.NF.)

R.NF.01. Usabilidad
Como usuario,
quiero una interfaz moderna, clara y fácil de usar,
para realizar mis alquileres de forma ágil desde el móvil.

R.NF.02. Seguridad
Como sistema,
quiero cifrar la información personal y financiera,
para proteger los datos de los usuarios.

R.NF.03. Disponibilidad
Como empresa,
quiero que el sistema esté disponible el 99% del tiempo,
para garantizar un servicio continuo.

R.NF.04. Rendimiento
Como usuario,
quiero que las consultas y operaciones respondan en pocos segundos,
para disfrutar de una experiencia fluida.

R.NF.05. Escalabilidad
Como empresa,
quiero que el sistema soporte un crecimiento de usuarios y vehículos,
para adaptarse a la expansión del negocio.

R.NF.06. Compatibilidad
Como usuario,
quiero acceder al sistema desde distintos dispositivos y navegadores,
para usarlo sin restricciones.

R.NF.07. Mantenibilidad
Como administrador del sistema,
quiero que el software permita actualizaciones sin afectar la operación,
para garantizar su evolución a largo plazo.
